## üì¶ Getting Started

### Prerequisites
* Flutter SDK (Stable channel)
* Android Studio / Xcode (for mobile deployment)

### Installation
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/ClawShelf/ClawShelf.git
    ```
2.  **Install dependencies**:
    ```bash
    cd ClawShelf
    flutter pub get
    ```
3.  **Run the application**:
    ```bash
    flutter run
    ```
